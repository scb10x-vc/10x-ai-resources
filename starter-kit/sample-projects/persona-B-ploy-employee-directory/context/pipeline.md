# Pipeline

1. Maintain `employees.csv` by adding or updating rows when people join, move teams, or change roles.
2. Open Codex and attach `employees.csv`, `report-starter.css`, and `skills/skill-csv-to-directory.md`.
3. Paste the prompt from `GETTING-STARTED.md`.
4. Codex generates `employee-directory.html`.
5. Open `employee-directory.html` in any browser and review layout and content.
6. Share the HTML file by email, internal chat, or a shared drive so anyone can open it in a browser.
