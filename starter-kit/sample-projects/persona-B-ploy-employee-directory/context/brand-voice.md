# Brand Voice

Write in a warm, casual, genuinely friendly tone. This is internal culture content, not a corporate memo.

Use first names in running text. Do not use Mr., Ms., or overly formal honorific phrasing.

Let personality come through. Hobbies and fun facts should sound human and natural, not like a rigid form field.

Keep sentences short and clear. Use plain language that any teammate can skim quickly.

Avoid formal titles in paragraph text, bullet points that read like job descriptions, and any wording that sounds like an HR policy document.
