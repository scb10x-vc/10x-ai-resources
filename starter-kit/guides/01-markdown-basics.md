# What is Markdown?

## The one-sentence version
Markdown is a plain text file with a few simple symbols that tell AI and other tools how to format things.

## Why this matters for your work
Codex reads and writes markdown naturally. When your content is in `.md` format, you can ask Codex to reorganize sections, clean up structure, or turn the draft into a finished document without manual copy and paste.

## The cheatsheet
| What you type | What it becomes |
|---|---|
| `# Heading 1` | Large title |
| `## Heading 2` | Section heading |
| `**bold text**` | Bold text |
| `- bullet point` | Bullet list item |
| `> This is a quote` | Highlighted or indented block |
| ````` | A formatted code or data block |
| `[link text](URL)` | Clickable link |

## You do not need a special app to write it
Any text editor works: Notepad on Windows, TextEdit on Mac, or Google Docs if you export afterward. You can also type directly in Codex and ask it to save the result as a `.md` file.

## A note on file extensions
A markdown file ends with `.md`. If someone sends `notes.md`, open it in any text editor and you will see plain text with the symbols shown above.

## TL;DR
- Markdown is a plain text format that AI reads and writes natively
- A small set of symbols (like # and **) control all the formatting
- You do not need any special app to read or write it
